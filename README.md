# moonshine 🌙

> **missing declarative package manager, that sits in-between actual managers and you to provide one interface for all**

## install

1. with `go install`:
```shell
go install pyrorhythm.dev/moonshine/cmd/moonshine@latest
```
2. with `brew`:
```shell
brew tap pyrorhythm/tap
brew install moonshine
```
> when installing with `brew`, `moon` alias is also installed.
